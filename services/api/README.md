# @links/api

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start:dev

# production mode
$ npm run start:prod
```

## Prisma

```bash
# builds required objects
$ yarn prisma generate
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
